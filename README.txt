Проект сайта психолога Поповой Натальи Николаевны

Содержимое:
- index.html — основной сайт
- server.js — backend для отправки заявок по email
- package.json — зависимости Node.js
- .env.example — пример настроек окружения
- images/mom1.jpg и images/mom2.jpg — фотографии

Как запустить локально:
1. Установите Node.js
2. Откройте терминал в папке проекта
3. Выполните: npm install
4. Скопируйте .env.example в .env
5. Заполните SMTP_PASS реальным паролем
6. Выполните: node server.js
7. Откройте в браузере: http://localhost:3000

Для хостинга нужен сервер с поддержкой Node.js.
